jTree: Family Tree Maker Script

It is a complete script for building professional solution for users to create their own family tree in real time using Jquery, Bootstrap & Php as core platform.

Script can allow you to

  • Create dynamic family trees in real time.
  • Save tree, nodes (person info) and relationships.
  • Load tree from database
  • Display information in tabular format for admin use.
  • and more.
